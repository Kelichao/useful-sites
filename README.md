# 笔试题型总结
- 马上就要过年了，而过完年，马上便是各大公司的校招，各个还在学校的童鞋也都要出来找实习单位了，那么如何让我们在面试以及笔试中脱颖而出呢？答案之一便是：多刷题，刷好题。下面我分别总结了（html/css进阶篇 ） （javascript进阶篇）相信可以作为大家升职加薪提供筹码。
